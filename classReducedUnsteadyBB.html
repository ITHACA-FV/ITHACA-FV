Detailed Description

Class where it is implemented a reduced problem for the unsteady Navier-stokes problem.

In this class are implemented the methods to solve a reduced version of the stationary Navier-stokes equations. This class must be used together with the steadyNS class

Examples:
10UnsteadyBBEnclosed.C, and 11UnsteadyBBOpen.C.

Method to perform an online solve using a PPE stabilisation method.

Parameters

[in]vel_now_BCThe scalar of online velocity. It is defined in with an Eigen::MatrixXd and must have one col and as many rows as the number of parametrized boundary conditions.
[in]NParaSetThe parameter sets number for which the online solution is solved.
[in]startSnapThe first snapshot taken from the offline snapshots and used to get the reduced initial condition default = 0.
