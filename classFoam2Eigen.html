Perform the projection of an OpenFOAM field onto a set of modes using the Eigen matrix multiplication.

It is the standard Galerkin projection of a field onto a reduced basis:

\[ \mathbf{f} \in R^{Nx1} , \mathbf{f_r} \in R^{nx1} , \mathbf{M} \in R^{Nxn} \]

\[ \mathbf{f_r} = \mathbf{M^T f} \]

Parameters

[in]modesThe M OpenFOAM modes contained in a PtrList
[in]NmodesThe n number of modes you want to use (The size of input modes can be bigger than the requested number of modes)
Template Parameters

type_PtrListthe type of the PtrList for the modes can be volScalarField or volVectorField
Returns
Eigen vector with the projected field

Perform the projection of an OpenFOAM fvMatrix onto a set of modes using the Eigen matrix multiplication.

It is performed the standard projection used in reduced order models to obtain the reduced matrices:

\[ \mathbf{A} \in R^{NxN} , \mathbf{A_r} \in R^{nxn} , \mathbf{b} \in R^{Nx1} , \mathbf{b_r} \in R^{nx1} , \mathbf{M} \in R^{Nxn} \]

\[ \mathbf{A_r} = \mathbf{M^T A M} , \mathbf{b_r} = \mathbf{M^T b} \]

Parameters
[in]matrixThe A OpenFOAM fvMatrix
[in]modesThe M OpenFOAM modes contained in a PtrList
[in]NmodesThe n number of modes you want to use (The size of input modes can be bigger than the requested number of modes)
Template Parameters
type_mtype of fvMatrix can be either scalar or vectorial
type_PtrListthe type of the PtrList for the modes can be volScalarField or volVectorField
Returns
A tuple containing the projected Ar matrix and vector br which are coming from the source term.
