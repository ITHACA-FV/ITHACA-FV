Detailed Description

Implementation of a parametrized full order unsteady NS problem weakly coupled with the energy equation and preparation of the the reduced matrices for the online solve.

In this class are implemented the methods for the offline solve of a unsteady NST problem and the for the generation of the reduced matrices for subsequent online solve, this class is a son of the unsteadyNS class
