Compute the bases or read them for a vector field.

Parameters
[in]snapshotsUa PtrList of volVectorField snapshots.
[out]modesUa PtrList of volVectorField where the modes are stored (it must be passed empty).
[in]podexboolean variable 1 if the modes have been already computed and stored (in this case the function is reading them) 0 elsewhere.
[in]supexboolean variable 1 if the supremizer modes have been already computed and stored (in this case the function is reading them) 0 elsewhere.
[in]supboolean variable 1 if you want to compute the supremizer modes 0 elsewhere.
[in]nmodesint variable to set the number of modes to be stored, if set to 0 the maximum number of modes will computed.
Examples:
02thermalBlock.C, 03steadyNS.C, and 04unsteadyNS.C.
