Detailed Description

Class where it is implemented a reduced problem for the unsteady Navier-stokes weakly coupled with the energy equation problem.

In this class are implemented the methods to solve a reduced version of the non-stationary Navier-stokes coupled with the energy equations. This class must be used together with the unsteadyNST class

Method to perform an online solve using a supremizer stabilisation method.

Parameters
[in]vel_nowThe vector of online velocity. It is defined in with an Eigen::MatrixXd and must have one col and as many rows as the number of parametrized boundary conditions.
[in]temp_nowThe vector of online temperature. It is defined in with an Eigen::MatrixXd and must have one col and as many rows as the number of parametrized boundary conditions.
[in]startSnapThe first snapshot taken from the offline snahpshots and used to get the reduced initial condition.
